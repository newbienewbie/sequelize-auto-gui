# Sequelize-auto-gui

[Sequelize-Auto](https://github.com/sequelize/sequelize-auto) GUI tool

## install

```
> git clone https://github.com/newbienewbie/sequelize-auto-gui
> cd sequelize-auto-gui
> npm install
> npm run webpack  -- -p
```


## run 

```
> npm run start
```
